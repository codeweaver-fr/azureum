# Sprint 001 — Fondation et gouvernance

## Statut

**Verrouillé**

## Objectif

Créer la fondation documentaire et méthodologique du nouveau projet AZUREUM.

Ce Sprint ne contient aucune implémentation applicative.

## Périmètre

- initialisation du dépôt Git ;
- création de l'arborescence documentaire ;

- protection des secrets et fichiers générés.

## Critères d'acceptation

- le dépôt Git est initialisé ;
- la branche principale est `main` ;
- aucun secret n'est suivi ;
- aucun code applicatif n'existe ;
- les règles de gouvernance sont explicites ;
- les périmètres V1 et V2 sont distingués ;
- l'état du projet est compréhensible sans contexte extérieur ;
- `git diff --check` réussit ;
- le dépôt est propre après le commit.
