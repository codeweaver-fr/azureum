# Glossaire métier

Ce document définit les termes utilisés dans le projet AZUREUM.

Aucun terme métier ne doit être utilisé dans le code sans définition validée.

## Termes à définir

- Artiste
- Œuvre
- Collection
- Membre
- Acheteur
- Commande
- Paiement
- Certificat d'authenticité
- Administrateur
- Enchère
- Offre
