# Roadmap produit

## V1 — Galerie premium mono-artiste

La V1 doit permettre de présenter et vendre les œuvres d'un seul artiste dans un environnement premium.

Le périmètre précis sera défini pendant les Sprints produit.

## V2 — Enchères

La V2 pourra introduire un moteur d'enchères complet inspiré du fonctionnement d'eBay, sans reproduire son identité ni son interface.

Les règles d'enchères feront l'objet d'une phase de conception dédiée.

## Hors périmètre actuel

- marketplace ouverte à plusieurs artistes ;
- application mobile native ;
- réseau social ;
- NFT ;
- cryptomonnaies ;
- enchères dans la V1.
